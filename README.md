# OpenSciFlow White Paper

Position paper drafts for open, modular, local-first AI for Science workflows.

Working title:

> Toward Open, Modular, and Local-First AI for Science Workflows

## Status

Draft. Do not present this as a mature system evaluation.

## Core argument

AI for Science does not only need stronger models or more autonomous agents. It also needs open conventions for:

- tool/model metadata;
- workflow templates;
- local/HPC execution;
- validation;
- license and citation propagation;
- reproducibility records;
- explicit limitations.
